# /thinkmap — Cartografiere stil de gândire

Construiește un ThinkMap pe 14 dimensiuni cognitive: structură logică, paradigmă de interpretare, metafore recurente, tipare de argumentare, valori dominante și altele. Funcționează în trei moduri: interviu live, analiză transcript sau ambele combinate.

## Când îl folosești

- Vrei să înțelegi cum gândești și comunici
- Analizezi stilul de gândire al unui colaborator sau client din transcrieri
- Pregătești materiale de comunicare personalizate pe un anumit profil cognitiv

## Instalare

```bash
cp skill.md ~/.claude/commands/thinkmap.md
```

Activare în Claude Code: `/thinkmap`
