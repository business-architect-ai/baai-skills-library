# /book-end — Încheie sesiunea de lectură

Folosit la finalul unei sesiuni de lectură, research sau documentare. Extrage 3-5 idei principale, citate memorabile, conexiuni cu alte concepte și acțiuni concrete. Salvează totul structurat în `~/Documents/reading-actions.md`.

## Când îl folosești

- Ai terminat de citit o carte, articol sau documentație
- Vrei să capturezi ce ai reținut înainte să uiți
- Construiești o bază de cunoștințe personală din lecturile tale

## Instalare

```bash
cp skill.md ~/.claude/commands/book-end.md
```

Activare în Claude Code: `/book-end`
