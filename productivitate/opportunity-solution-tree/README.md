# /opportunity-solution-tree - Product discovery tree

Construieste un Opportunity Solution Tree pentru product discovery: outcome masurabil, oportunitati din research, solutii posibile si experimente de validare.

## Cand il folosesti

- Vrei sa decizi ce sa construiesti.
- Ai feedback/research si vrei sa-l transformi in roadmap.
- Echipa sare prea repede la features.
- Vrei sa conectezi outcome-uri la oportunitati si experimente.

## Ce produce

- Desired outcome
- Evidence base
- Opportunities cu scoring
- Solutions
- Experiments
- Tree view
- Recomandare de discovery

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

```bash
cp productivitate/opportunity-solution-tree/skill.md ~/.claude/commands/opportunity-solution-tree.md
```
