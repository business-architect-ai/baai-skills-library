# /research-users - User research synthesis

Sintetizeaza research de utilizatori: interviuri, survey-uri, feedback, support tickets, analytics sau descriere de produs. Produce personas, segmente comportamentale, customer journey map, insighturi si recomandari.

## Cand il folosesti

- Ai date brute despre utilizatori.
- Vrei personas sau segmente comportamentale.
- Vrei customer journey map.
- Vrei insighturi pentru produs, marketing, onboarding sau roadmap.

## Ce produce

- Data snapshot
- Personas
- Behavioral segments
- Customer journey map
- Key insights cu confidence
- Recommendations
- Follow-up research plan
