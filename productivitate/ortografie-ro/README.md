# /ortografie-ro — Ortografie română (sentence case)

Aplică regulile corecte de capitalizare pentru limba română pe fișiere sau text: sentence case obligatoriu în titluri, interdicție title case, literă mică după două puncte. Afișează modificările propuse înainte de a le aplica.

## Când îl folosești

- Ai un document, curs sau site cu capitalizare incorectă
- Generezi conținut în română și vrei să respecți normele
- Faci review la materiale înainte de publicare

## Instalare

```bash
cp skill.md ~/.claude/commands/ortografie-ro.md
```

Activare în Claude Code: `/ortografie-ro` (pe textul curent) sau `/ortografie-ro cale/fisier.md`
