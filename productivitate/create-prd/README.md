# /create-prd — Product Requirements Document complet

Creeaza un PRD complet pentru un produs, feature sau initiativa: problema, obiective, segmente, value proposition, scope, requirements, assumptions, riscuri si release plan.

## Cand il folosesti

- Vrei un PRD pentru un feature nou.
- Vrei sa transformi o idee de produs intr-o specificatie clara.
- Ai nevoie de handoff catre dev/design/leadership.
- Vrei sa clarifici ce intra in v1 si ce ramane pentru mai tarziu.

## Diferenta fata de `/plan`

- `/plan` = mini-spec rapid inainte de cod.
- `/create-prd` = document de produs complet pentru aliniere intre echipe.

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

```bash
cp productivitate/create-prd/skill.md ~/.claude/commands/create-prd.md
```
