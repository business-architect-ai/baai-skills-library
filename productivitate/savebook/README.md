# /savebook — Salvează o carte sau resursă

Adaugă o carte, articol sau resursă în lista ta de lectură (`~/Documents/reading-list.md`). Caută automat autorul și categoria, verifică dacă există deja și confirmă salvarea. Acceptă titlu, autor sau URL.

## Când îl folosești

- Dai peste o carte sau articol pe care vrei să nu-l uiți
- Construiești o listă de lectură pe categorii (AI, Business, Marketing etc.)

## Instalare

```bash
cp skill.md ~/.claude/commands/savebook.md
```

Activare în Claude Code: `/savebook Titlul Cartii` sau `/savebook https://...`
