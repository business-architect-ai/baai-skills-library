# /huashu-design — Prototipuri și design HTML hi-fi

Creează prototipuri interactive, slide-uri, animații și mockup-uri UI direct în HTML. Embodiază diferiți specialiști în funcție de task: UX designer, animator, slide designer sau prototyper. Suportă comenzi în română, engleză, chineză și rusă.

Nu folosești pentru ad creatives Facebook/Instagram, ci pentru prototipuri, demo-uri interactive și design exploration.

## Când îl folosești

- Vrei un prototip rapid de interfață sau flow
- Creezi slide-uri cu animații pentru o prezentare
- Explorezi variante de design înainte de implementare
- Faci un demo interactiv pentru un client

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/huashu-design
cp skill.md ~/.claude/skills/huashu-design/SKILL.md
```

Activare în Claude Code: `/huashu-design` sau `fă prototip pentru...`
