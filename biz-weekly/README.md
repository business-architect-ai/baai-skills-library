# /biz-weekly — Retrospectivă săptămânală

Review săptămânal structurat: ce ai realizat, ce a mers bine, ce nu a mers și de ce, ce ai învățat, ce activități ți-au dat sau luat energie și top 3 priorități pentru săptămâna următoare cu scor și trend.

## Când îl folosești

- Faci review săptămânal de business sau productivitate
- Vrei să identifici pattern-uri în ce merge și ce nu
- Pregătești priorități pentru săptămâna următoare

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-weekly.md
```

Activare în Claude Code: `/biz-weekly`
