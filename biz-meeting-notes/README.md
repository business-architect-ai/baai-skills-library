# /biz-meeting-notes — Note post-întâlnire

Transformă notele brute dintr-o întâlnire în minute structurate: decizii luate, acțiuni cu responsabil și deadline, întrebări deschise și un draft de email de follow-up gata de trimis.

## Când îl folosești

- Ai terminat o întâlnire și vrei să clarifici ce urmează
- Trimiți un follow-up echipei sau clientului
- Documentezi o decizie importantă

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-meeting-notes.md
```

Activare în Claude Code: `/biz-meeting-notes`
