# /biz-ops — Audit procese operaționale

Mapează un proces din business-ul tău pas cu pas, identifică bottleneck-urile și risipa, și produce un plan de optimizare ordonat după prioritate: ce elimini, ce automatizezi, ce simplifici, cu economii calculate în timp și bani.

## Când îl folosești

- Ai un proces care te frustrează sau ia prea mult timp
- Vrei să automatizezi ceva dar nu știi de unde să începi
- Simți că faci aceleași lucruri manual de prea multe ori
- Pregătești delegarea sau documentarea unui proces

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-ops.md
```

Activare în Claude Code: `/biz-ops`
