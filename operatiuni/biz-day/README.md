# /biz-day — Ritual de dimineață pentru antreprenori

Pornește ziua cu context clar: priorități, KPIs actuali, follow-up-uri promise și deadlines apropiate. Afișează și un toolkit rapid cu toate skillurile de business disponibile. De la a doua rulare, știe deja contextul tău.

## Când îl folosești

- Dimineața, înainte să începi lucrul
- Vrei să intri în zi cu cap limpede, nu reactiv
- Folosit regulat devine un ritual de claritate zilnică

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-day.md
```

Activare în Claude Code: `/biz-day`
