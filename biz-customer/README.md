# /biz-customer — Avatar client

Construiește un profil detaliat al clientului ideal: cine e, ce îl doare, ce vrea, cum decide, ce limbaj folosește și unde îl găsești. Dacă ai feedback-uri sau conversații reale cu clienți, le analizează pentru limbaj autentic.

## Când îl folosești

- Nu ești sigur cui te adresezi exact
- Copyul tău nu convertește și vrei să înțelegi de ce
- Lansezi un produs nou și vrei să validezi piața
- Vrei să segmentezi audiența pentru campanii diferite

```bash
cp skill.md ~/.claude/commands/biz-customer.md
```
