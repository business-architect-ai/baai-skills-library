# /biz-copy — Copywriting pe vocea brandului

Scrie copy pentru orice canal (email, landing page, ad, social post, sales page) pe baza avatarului clientului tău. Produce întotdeauna 2-3 variante de hook sau headline cu recomandare explicită.

## Când îl folosești

- Ai nevoie de un email de vânzări sau newsletter
- Scrii o pagină de landing sau o pagină de vânzare
- Creezi un ad pentru social media
- Vrei variante de headline pentru același conținut

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-copy.md
```

Activare în Claude Code: `/biz-copy`
