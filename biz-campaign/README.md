# /biz-campaign — Planificare campanie de marketing

Construiește un plan complet de campanie: obiectiv, audiență, mesaj principal, canale cu acțiuni și KPIs, timeline săptămânal și plan de contingență pentru ce nu merge conform planului.

## Când îl folosești

- Lansezi un produs sau serviciu nou
- Vrei să crești leads sau vânzări pe o perioadă definită
- Planifici o campanie sezonieră
- Organizezi un eveniment sau webinar

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-campaign.md
```

Activare în Claude Code: `/biz-campaign`
