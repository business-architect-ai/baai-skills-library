# /xlsx — Analizează fișiere Excel sau CSV

Citește un fișier Excel sau CSV, afișează un rezumat (coloane, preview, statistici de bază) și întreabă ce vrei să faci: analiză, filtrare, calcule noi sau export. Nu modifică fișierul original fără confirmare.

## Când îl folosești

- Ai un raport de vânzări, un export din CRM sau orice tabel de date
- Vrei statistici rapide fără să deschizi Excel
- Filtrezi sau transformi date pentru un raport

## Instalare

```bash
cp skill.md ~/.claude/commands/xlsx.md
```

Activare în Claude Code: `/xlsx cale/catre/fisier.xlsx`
