# /biz-decision — Framework de decizie structurat

Te ajută să analizezi orice decizie dificilă: mapează opțiunile, estimează pro/contra și efectele de ordin 2, aplică filtre rapide (Regula Bezos, reversibilitate) și oferă o recomandare clară cu primul pas concret.

## Când îl folosești

- Ai o decizie grea și te tot învârți în jurul ei
- Vrei să compari 2-3 opțiuni în mod obiectiv
- Ai termen limită și nu poți amâna

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-decision.md
```

Activare în Claude Code: `/biz-decision`
