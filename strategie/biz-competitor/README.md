# /biz-competitor — Analiză competitivă

Cercetează competitorii tăi (site, prețuri, reviews, social media), construiește o hartă comparativă și identifică vulnerabilitățile lor, spațiile albe din piață și acțiunile concrete pe care le poți lua.

## Când îl folosești

- Intri pe o piață nouă și vrei să înțelegi peisajul
- Simți că pierzi clienți în favoarea unui competitor
- Vrei să îți repoziționezi oferta
- Pregătești o strategie de diferențiere

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-competitor.md
```

Activare în Claude Code: `/biz-competitor`
