# /sales-playbook - Playbook de vanzari

Construieste un sales playbook complet pentru o miscare comerciala: audienta, ICP, mesaj, discovery, talk tracks, battlecards, obiectii, asseturi, rollout, coaching si KPI-uri.

## Cand il folosesti

- Lansezi o oferta, produs sau miscare GTM.
- Vrei sa standardizezi cum vinde echipa.
- Pregatesti SDR, AE, CSM, partners sau founder-led sales.
- Vrei sa unesti messaging, battlecards, call briefs si objection handling intr-un sistem operational.

## Ce produce

- Motion definition
- ICP & buying committee
- Value narrative
- Sales process
- Discovery guide
- Talk tracks
- Objection handling matrix
- Competitive notes
- Asset checklist
- Enablement rollout
- Coaching & reinforcement plan
- Metrics si risks/gaps
