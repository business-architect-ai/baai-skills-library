# /call-brief-framework - Call brief pentru sales

Pregateste un call brief concis pentru intalniri de vanzari, discovery, demo, negociere sau exec alignment: obiectiv, stakeholderi, agenda, roluri, mesaje cheie, intrebari, proof points, riscuri si follow-up.

## Cand il folosesti

- Pregatesti un discovery call, demo, negociere sau executive meeting.
- Ai mai multi participanti si trebuie aliniate roluri.
- Vrei agenda, talk track, intrebari si proof points.
- Exista risc de deal: competitor, buget, legal, procurement sau stakeholder absent.

## Ce produce

- Meeting snapshot
- Participants & roles
- Agenda cu owner si checkpoint
- Key messages & proof
- Discovery questions
- Objection prep
- Risks & watchouts
- Follow-up log
- Manager scorecard
