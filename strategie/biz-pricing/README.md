# /biz-pricing — Review strategie de preț

Analizează prețul tău raportat la valoarea oferită și la piață, identifică pragurile psihologice relevante și produce o recomandare clară cu mod de implementare și testare, fără salturi bruște.

## Când îl folosești

- Nu ești sigur dacă prețul tău e corect
- Vrei să crești prețul dar nu știi cum să o faci
- Lansezi un produs nou și stabilești prețul de la zero
- Restructurezi planurile de abonament

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-pricing.md
```

Activare în Claude Code: `/biz-pricing`
