# /biz-review-portable - Diagnostic business portabil

Diagnostic complet de business fara dependente de memoria Claude Code: model de business, unit economics, moat, crestere, riscuri, sanatate generala si recomandari pe 30 zile.

## Cand il folosesti

- Vrei diagnostic de business.
- Vrei sa intelegi modelul, cresterea, riscurile si prioritatile.
- Lucrezi in Codex sau in Claude Code fara `LEARNING ENGINE`.

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

## Instalare Claude Code

```bash
cp strategie/biz-review-portable/skill.md ~/.claude/commands/biz-review-portable.md
```

## Instalare Codex

```bash
mkdir -p ~/.codex/skills/biz-review-portable
cp strategie/biz-review-portable/skill.md ~/.codex/skills/biz-review-portable/SKILL.md
```
