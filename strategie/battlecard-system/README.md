# /battlecard-system - Sistem de battlecard competitiv

Construieste battlecard-uri competitive pentru sales, marketing si leadership: pozitionare fata de competitor, diferentiatori, proof points, talk tracks, obiectii, win/loss signals si reguli de folosire.

## Cand il folosesti

- Pregatesti echipa pentru un competitor concret.
- Ai nevoie de objection handling si talk track pentru sales.
- Vrei sa transformi win/loss notes intr-un asset operational.
- Lansezi update de produs, pricing sau pozitionare competitiva.

## Ce produce

- Competitor snapshot
- Permission to win
- Differentiators cu proof
- Talk track
- Objection handling matrix
- Trap questions
- Win/loss signals
- One-page version pentru sales

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

