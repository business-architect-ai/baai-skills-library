# /plan — Planifică un feature înainte de cod

Ghidează procesul de planificare a unui feature nou: cine sunt utilizatorii, ce pot face, care e flow-ul principal, ce poate merge prost și cum se leagă de restul aplicației. Produce un mini-spec confirmat înainte de a scrie orice cod.

## Când îl folosești

- Ești pe cale să implementezi ceva și vrei să te asiguri că ai gândit toate cazurile
- Lucrezi cu un freelancer sau AI și vrei să documentezi spec-ul clar
- Planifici un sprint sau o funcționalitate nouă

## Instalare

```bash
cp skill.md ~/.claude/commands/plan.md
```

Activare în Claude Code: `/plan`
