# /waapi — Animații cu Web Animations API

Implementează animații folosind Web Animations API (WAAPI), API-ul nativ al browserului pentru animații JavaScript, fără dependențe externe.

## Când îl folosești

- Vrei animații JavaScript fără librării externe
- Ai nevoie de control programatic complet asupra animațiilor
- Construiești componente refolosibile cu animații native

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/waapi
cp skill.md ~/.claude/skills/waapi/SKILL.md
```
