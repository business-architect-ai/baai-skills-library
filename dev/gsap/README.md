# /gsap — Animații cu GSAP (GreenSock)

Implementează animații performante cu GSAP, cel mai folosit framework de animații JavaScript. Acoperă Tweens, Timelines, ScrollTrigger și alte plugin-uri GSAP.

## Când îl folosești

- Ai nevoie de animații complexe cu control precis
- Construiești experiențe cu animații declanșate de scroll
- Animezi SVG-uri sau elemente complexe

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/gsap
cp skill.md ~/.claude/skills/gsap/SKILL.md
```
