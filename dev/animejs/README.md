# /animejs — Animații cu Anime.js

Implementează animații web folosind biblioteca Anime.js. Ghidează selecția tipului de animație, parametrilor (durată, easing, delay) și integrarea în proiect.

## Când îl folosești

- Adaugi animații la elemente HTML/SVG în proiectul tău
- Vrei animații declanșate de scroll, hover sau click
- Creezi timeline-uri de animații complexe

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/animejs
cp skill.md ~/.claude/skills/animejs/SKILL.md
```
