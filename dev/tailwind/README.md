# /tailwind — Stilizare cu Tailwind CSS

Implementează UI cu Tailwind CSS: clase utilitare, responsive design, dark mode, componente și configurare `tailwind.config`. Acoperă Tailwind v3 și v4.

## Când îl folosești

- Construiești sau stilizezi componente cu Tailwind
- Configurezi un proiect nou cu Tailwind
- Ai nevoie de layout responsive sau teme custom

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/tailwind
cp skill.md ~/.claude/skills/tailwind/SKILL.md
```
