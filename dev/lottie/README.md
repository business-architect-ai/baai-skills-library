# /lottie — Animații Lottie

Integrează și controlează animații Lottie (fișiere JSON exportate din Adobe After Effects) în aplicații web. Acoperă redare, control viteză, loop și interactivitate.

## Când îl folosești

- Ai un fișier `.json` Lottie de integrat în site sau aplicație
- Vrei animații complexe exportate din After Effects
- Adaugi ilustrații animate sau iconițe animate

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/lottie
cp skill.md ~/.claude/skills/lottie/SKILL.md
```
