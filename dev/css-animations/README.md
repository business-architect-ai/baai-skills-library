# /css-animations — Animații CSS

Creează animații CSS cu `@keyframes`, `transition` și `animation`. Acoperă pattern-uri comune: fade, slide, bounce, spin, și animații declanșate de clase sau stări.

## Când îl folosești

- Adaugi animații simple fără JavaScript
- Vrei micro-interacțiuni pe butoane, carduri sau meniuri
- Optimizezi performanța animațiilor (GPU-accelerated)

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/css-animations
cp skill.md ~/.claude/skills/css-animations/SKILL.md
```
