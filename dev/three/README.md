# /three — Grafică 3D cu Three.js

Creează scene 3D interactive în browser cu Three.js: geometrii, materiale, lumini, camere, animații și încărcare de modele 3D (GLTF/GLB).

## Când îl folosești

- Construiești o scenă 3D sau o experiență interactivă web
- Adaugi un model 3D animat pe un site
- Creezi vizualizări de date sau produse în 3D

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/three
cp skill.md ~/.claude/skills/three/SKILL.md
```
