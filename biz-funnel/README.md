# /biz-funnel — Diagnostic funnel de conversie

Mapează pașii funnelului tău, identifică unde se pierd oamenii și de ce, compară cu benchmark-urile din industrie și produce fix-uri prioritizate după impact, cu mod de testare pentru fiecare.

## Când îl folosești

- Ai trafic dar puține vânzări
- Știi că se pierd oameni undeva dar nu știi unde exact
- Vrei să optimizezi o etapă specifică din funnel
- Verifici sănătatea unui funnel înainte de o campanie

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-funnel.md
```

Activare în Claude Code: `/biz-funnel`
