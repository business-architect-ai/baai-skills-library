# /push — Trimite modificările pe GitHub

Verifică starea branch-ului, avertizează dacă există modificări necommit-uite și execută `git push` cu confirmare explicită. Afișează link-ul către repo după succes.

## Când îl folosești

- Ai făcut commit local și vrei să trimiți pe GitHub
- Vrei să te asiguri că branch-ul e configurat corect înainte de push

## Instalare

```bash
cp skill.md ~/.claude/commands/push.md
```

Activare în Claude Code: `/push`
