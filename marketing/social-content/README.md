# /social-content — Conținut social media pe orice platformă

Creează, planifică și optimizează conținut social media pentru LinkedIn, Twitter/X, Instagram, TikTok, Facebook. Include formule de hook, sistem de repurposing și strategie de engagement zilnic.

## Când îl folosești

- Ai nevoie de postări pentru una sau mai multe platforme sociale
- Vrei un calendar de conținut pe săptămână sau lună
- Ai conținut existent de repurposat pe mai multe canale

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/social-content
cp skill.md ~/.claude/skills/social-content/SKILL.md
```
