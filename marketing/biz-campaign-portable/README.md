# /biz-campaign-portable - Plan campanie portabil

Planifica o campanie de marketing cu obiectiv, audienta, mesaj, canale, actiuni, calendar, KPI-uri si contingency plan.

## Cand il folosesti

- Vrei plan de campanie pentru vanzari, leads, awareness sau lansare.
- Ai nevoie de canale, calendar si KPI-uri.
- Lucrezi in Codex sau in Claude Code fara `LEARNING ENGINE`.

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

## Instalare Claude Code

```bash
cp marketing/biz-campaign-portable/skill.md ~/.claude/commands/biz-campaign-portable.md
```

## Instalare Codex

```bash
mkdir -p ~/.codex/skills/biz-campaign-portable
cp marketing/biz-campaign-portable/skill.md ~/.codex/skills/biz-campaign-portable/SKILL.md
```

Versiunea originala [biz-campaign](../biz-campaign/) ramane `claude-code-only` si pastreaza `LEARNING ENGINE`.
