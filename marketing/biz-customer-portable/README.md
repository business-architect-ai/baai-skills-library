# /biz-customer-portable - Avatar client portabil

Construieste avatar client si customer lens din date disponibile: produs, client ideal, problema, trigger, criterii de cumparare, limbaj real, obiectii si canale.

## Cand il folosesti

- Vrei avatar client pentru oferta/produs.
- Vrei limbajul natural al clientului.
- Lucrezi in Codex sau in Claude Code fara `LEARNING ENGINE`.

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

## Instalare Claude Code

```bash
cp marketing/biz-customer-portable/skill.md ~/.claude/commands/biz-customer-portable.md
```

## Instalare Codex

```bash
mkdir -p ~/.codex/skills/biz-customer-portable
cp marketing/biz-customer-portable/skill.md ~/.codex/skills/biz-customer-portable/SKILL.md
```
