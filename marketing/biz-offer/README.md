# /biz-offer — Construiește sau revizuiește oferta

Structurează oferta ta cu toate elementele: core, bonusuri cu valoare percepută, garanție, urgență/scarcitate și CTA. Produce și un stack vizual gata de folosit în materiale de vânzări, plus răspunsuri la obiecțiile frecvente.

## Când îl folosești

- Oferta ta nu convertește și nu știi de ce
- Lansezi un produs nou și construiești oferta de la zero
- Vrei să adaugi bonusuri relevante care cresc valoarea percepută
- Pregătești o pagină de vânzare sau o prezentare

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-offer.md
```

Activare în Claude Code: `/biz-offer`
