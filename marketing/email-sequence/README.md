# /email-sequence — Creează secvențe de email automate

Construiește sau optimizează secvențe de email: welcome, nurture, re-engagement, post-purchase, educational, sales. Produce draft-uri complete cu subject, preview text, body și CTA pentru fiecare email.

## Când îl folosești

- Vrei să creezi un welcome sequence sau onboarding flow
- Ai nevoie de o secvență de nurture pentru leads noi
- Configurezi email automation (re-engagement, post-purchase)

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/email-sequence
cp skill.md ~/.claude/skills/email-sequence/SKILL.md
```
