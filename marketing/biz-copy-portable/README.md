# /biz-copy-portable - Copywriting portabil

Scrie copy specific pe canal si audienta, fara dependente de memoria Claude Code.

## Cand il folosesti

- Email, landing page, ad, social post, sales page sau CTA.
- Vrei variante de hook/headline.
- Lucrezi in Codex sau in Claude Code fara `LEARNING ENGINE`.

## Compatibilitate

```yaml
compatibility: codex-and-claude-code
```

## Instalare Claude Code

```bash
cp marketing/biz-copy-portable/skill.md ~/.claude/commands/biz-copy-portable.md
```

## Instalare Codex

```bash
mkdir -p ~/.codex/skills/biz-copy-portable
cp marketing/biz-copy-portable/skill.md ~/.codex/skills/biz-copy-portable/SKILL.md
```

## Nota

Versiunea originala [biz-copy](../biz-copy/) ramane `claude-code-only` si pastreaza `LEARNING ENGINE`.
