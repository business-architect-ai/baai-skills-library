# /content-strategy — Planifică strategia de conținut

Construiește o strategie de conținut bazată pe piloni, topic clusters și cercetare de client. Produce: piloni cu rationale, topicuri prioritizate cu keyword și buyer stage, calendar de conținut.

## Când îl folosești

- Nu știi ce conținut să creezi sau de unde să începi
- Vrei să construiești un plan pe piloni cu keyword targeting
- Ai conținut existent dar trafic scăzut și vrei un diagnostic

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/content-strategy
cp skill.md ~/.claude/skills/content-strategy/SKILL.md
```
