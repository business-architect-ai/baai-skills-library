# /message-architecture - Arhitectura de mesaj

Construieste un sistem coerent de messaging pentru produs, oferta, campanie sau brand: public, insight, promisiune, proof points, message house, hook bank, CTA matrix si adaptare pe canale.

## Cand il folosesti

- Vrei mesaje clare pentru website, email, LinkedIn, ads sau sales.
- Ai copy pe mai multe canale, dar mesajele nu se leaga intre ele.
- Pregatesti un launch, o oferta noua sau o repozitionare.
- Vrei brief pentru copywriter, agentie sau echipa de sales.

## Ce produce

- Audience & insight map
- Core promise
- Message house
- Proof library
- Hook bank
- CTA matrix
- Objections & responses
- Adaptare pe website, email, social si sales
