# /cold-email — Cold outreach B2B care primește reply-uri

Scrie, construiește și optimizează secvențe de cold email B2B. Acoperă: primul email, follow-up sequence (5-6 emailuri cu unghiuri diferite), optimizare pe baza performanței.

## Când îl folosești

- Vrei să scrii un prim email de cold outreach
- Ai nevoie de o secvență de follow-up cu unghiuri diferite
- Vrei să diagnostichezi și să repari o secvență cu performanță slabă

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/cold-email
cp skill.md ~/.claude/skills/cold-email/SKILL.md
```
