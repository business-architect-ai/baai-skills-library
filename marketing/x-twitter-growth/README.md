# /x-twitter-growth — Creștere pe X/Twitter

Motor de creștere X-specific: audit profil, inginerie thread-uri, strategie de reply, optimizare algoritm 2025-2026. Include cadențe de posting per dimensiune audiență și playbook de creștere săptămână cu săptămână.

## Când îl folosești

- Vrei să crești pe X/Twitter și nu știi de unde să începi
- Scrii tweet-uri sau thread-uri și vrei să optimizezi pentru algoritm
- Vrei să diagnostichezi de ce engagement-ul tău a scăzut

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/x-twitter-growth
cp skill.md ~/.claude/skills/x-twitter-growth/SKILL.md
```
