# /linkedin-content-planner — Plan de conținut LinkedIn pe 7 zile

Generează un plan săptămânal LinkedIn bazat pe disciplina celor 3 piloni (Authority 40-50%, Personal Narrative 30-40%, Community 20-30%). Include per-zi: pilonul, formatul, formula de hook, CTA, ora de publicare, ținte de comentarii.

## Când îl folosești

- Vrei să planifici o săptămână sau lună de conținut LinkedIn
- Vrei să ieși din modul ad-hoc și să stabilești un ritm
- Înainte de o săptămână de lansare (ai nevoie de aliniere pe pilonul product)

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/linkedin-content-planner
cp skill.md ~/.claude/skills/linkedin-content-planner/SKILL.md
```
