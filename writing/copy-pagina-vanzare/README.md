# /copy-pagina-vanzare — Copy complet pagină de vânzare

Generează tot copyul necesar pentru o pagină de vânzare prin 9 întrebări conversaționale puse una câte una. Produce un fișier `.md` structurat cu: hero, problemă, soluție, cum funcționează, beneficii, testimoniale, prețuri, FAQ, garanție și CTA final.

Unde lipsesc date reale (ex: testimoniale), marchează cu placeholder clar.

## Când îl folosești

- Lansezi un produs sau serviciu și construiești pagina de vânzare de la zero
- Vrei să structurezi copyul înainte de implementare în HTML sau no-code
- Faci un audit al paginii de vânzare curente

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/copy-pagina-vanzare
cp skill.md ~/.claude/skills/copy-pagina-vanzare/SKILL.md
```

Activare în Claude Code: `/copy-pagina-vanzare`
