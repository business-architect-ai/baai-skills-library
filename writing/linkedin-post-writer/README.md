# /linkedin-post-writer — Scrie postări LinkedIn cu formule de 2026

Creează postări LinkedIn de la zero folosind cele 10 formule de hook dovedite (anaphora, R.I.P., year-pivot, time-anchor, self-proving, paid-vs-free, curiosity-gap, odd-money, contrarian). Include pasul de humanizare automată.

## Când îl folosești

- Vrei să scrii o postare LinkedIn pe un subiect dat
- Ai un unghi și ai nevoie de un hook din formulele performante 2026
- Vrei să alegi dintr-un set de formate câștigătoare

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/linkedin-post-writer
cp skill.md ~/.claude/skills/linkedin-post-writer/SKILL.md
```
