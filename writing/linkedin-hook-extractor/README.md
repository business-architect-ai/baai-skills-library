# /linkedin-hook-extractor — Demontează hookul oricărei postări virale

Dă-i un URL sau textul unei postări virale de LinkedIn. Primești: ce formulă de hook folosește (din cele 10 canonice 2026), de ce a funcționat, și un template gol adaptat pe topicul tău.

## Când îl folosești

- Ai găsit o postare virală pe care vrei să o înțelegi și să o replici
- Vrei să studiezi pattern-urile unui creator
- Înainte de `/linkedin-post-writer`, ca să semezi draft-ul cu o structură dovedită

## Instalare

Acest skill este de tip plugin:

```bash
mkdir -p ~/.claude/skills/linkedin-hook-extractor
cp skill.md ~/.claude/skills/linkedin-hook-extractor/SKILL.md
```
