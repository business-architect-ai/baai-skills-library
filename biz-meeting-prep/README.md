# /biz-meeting-prep — Pregătire întâlnire

Pregătește o întâlnire cu agendă cronometrată, puncte de discuție, deciziile necesare, întrebările de pus și scenariile worst-case cu răspunsuri pregătite. Obiectivul întâlnirii trebuie să fie clar înainte să intri.

## Când îl folosești

- Ai o întâlnire importantă cu un client, partener sau investitor
- Vrei să intri pregătit, nu să improvizezi
- Ai o negociere sau o discuție dificilă de purtat

## Instalare

```bash
cp skill.md ~/.claude/commands/biz-meeting-prep.md
```

Activare în Claude Code: `/biz-meeting-prep`
