# /imagine — Generează o imagine

Trimite un prompt către agentul de generare imagini și produce rezultatul. Acceptă orice descriere text și opțional flag-ul `--model` pentru a selecta modelul de generare.

## Când îl folosești

- Vrei o imagine pentru un post, prezentare sau material de marketing
- Ai nevoie de un concept vizual rapid

## Instalare

```bash
cp skill.md ~/.claude/commands/imagine.md
```

Activare în Claude Code: `/imagine o fotografie cu un antreprenor la birou, lumină naturală`
