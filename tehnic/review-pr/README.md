# /review-pr — Review complet pull request

Lansează mai mulți agenți specializați în paralel pentru a revizui un PR: comentarii, acoperire teste, error handling, design tipuri și calitate generală a codului. Agregă rezultatele în categorii: critical, important, sugestii.

## Când îl folosești

- Ai un PR gata și vrei un review înainte de merge
- Vrei să verifici mai multe dimensiuni ale calității (teste, securitate, logică)
- Lucrezi solo și nu ai un coleg care să facă code review

## Instalare

```bash
cp skill.md ~/.claude/commands/review-pr.md
```

Activare în Claude Code: `/review-pr` sau `/review-pr tests errors`
