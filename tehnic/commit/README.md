# /commit — Salvează modificările în git

Ghidează procesul de commit: arată fișierele modificate, întreabă ce incluzi și ce mesaj vrei, execută `git add` și `git commit` cu format structurat. Nu face push automat.

## Când îl folosești

- Vrei să salvezi o etapă de lucru local
- Nu ești sigur ce fișiere să incluzi în commit
- Vrei un mesaj de commit bine structurat

## Instalare

```bash
cp skill.md ~/.claude/commands/commit.md
```

Activare în Claude Code: `/commit`
